Classpublic class Vector3D
InheritanceVector3D Object

Vector3D represents a vector or a point in three-dimensional cartesian coordinate space. The interface to this class is based on the interface to the flash.geom.Vector3D class included in Flash Player 10, but with some enhancements. This class has a number of methods that are not in the flash.geom.Vector3D class. Also, every method returns a reference to the Vector3D object so that a fluid programming style can be adopted for vector maths. For example v.incrementBy( u ).scaleBy( 7 ) A vector in the Vector3D class actually has four coordinates. The standard x, y, and z coordinates are used as normal for defining a location or a vector in 3D space. The forth coordinate (w) can be used in a number of ways. The most common use for the w cooordinate is to encapsulate the difference between a location and a vector. In this case, setting w to 1 indicates a location and setting it to 0 indicates a vector. This has a number of beneficial side effects, including
  • Adding a point and a vector produces a point. Adding two vectors produces a vector. Points can't be added together.
  • Subtracting a vector from a point produces a point. Subtracting a vector from a vector produces a vector. Subtracting a point from a point produces the vector from one point to the other. A point can't be subtracted from a vector.
  • Matrix transformations on vectors don't apply the translation elements. Matrix transformations on points do apply the translation elements.
The w coordinate may also be used to indicate a rotation, with w representing the angle of rotation and the x, y and z indicating the axis to rotate around. The w coordinate may be used to represent a projection transform for projecting a point onto a 2D surface.

wproperty 
public var w:Number

The w coordinate of the vector. The w coordinate may be used for a number of purposes. For example

  • Often, w coordinate of 0 indicates a vector, while a w coordinate of 1 usually represents a point. When transformed by a matrix, a Vector3D object with a zero w coordinate will be rotated and scaled but will not be translated.
  • The w coordinate may represent an angle of rotation about the vector represented by the x,y,z coordinates.
  • The w coordinate may represent a projection transform for projecting a point onto a 2D surface.
