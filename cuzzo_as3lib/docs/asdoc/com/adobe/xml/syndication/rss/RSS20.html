Class for parsing RSS 1.0 feeds. Note: the RSS20 class can be used to parse RSS versions 0.91, 0.92, and 2.0. It cannot be used to parse RSS 1.0. Use the RSS10 class instead.

See also

http://web.resource.org/rss/1.0/
