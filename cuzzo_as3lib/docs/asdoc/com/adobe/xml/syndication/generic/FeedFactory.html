Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 8.5

The FeedFactory allows you to create generic IFeed objects from any version of RSS or Atom. You can then access data within the feed without knowing what type of feed or which version it is. For most people, this will be the starting point for the XML syndication project.
