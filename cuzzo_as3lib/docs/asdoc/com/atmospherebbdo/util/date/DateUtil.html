daysInMonthsproperty
public static var daysInMonths:Array

The number of days appearing in each month. May be used for easy index lookups. The stored value for February corresponds to a standard year--not a leap year.

parseRFC822()method 
public static function parseRFC822(str:String):Date

Language Version : ActionScript 3.0
Runtime Versions : AIR 1.0, Flash Player 9.0

Parses dates that conform to RFC822 into Date objects. This method also supports four-digit years (not supported in RFC822), but two-digit years (referring to the 20th century) are fine, too. This function is useful for parsing RSS .91, .92, and 2.0 dates.

Parameters

str:String

Returns
Date

http://asg.web.cmu.edu/rfc/rfc822.html
