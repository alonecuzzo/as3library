asHexString()function
public function asHexString(number:uint, minimumLength:int = 1):String

Adapted from as3lib. Converts a uint into a string in the format "0x123456789ABCDEF". This function is quite useful for displaying hex colors as text. The function toString(radix:uint) does a similar job but doesn't include "0x" at the beginning of the result nor does it include leading zeroes.

Parameters

number:uint — The number to convert to hex. Note, numbers larger than 0xFFFFFFFF may produce unexpected results.
 
minimumLength:int (default = 1) — The smallest number of hexits to include in the output. Missing places will be filled in with 0's. e.g. getNumberAsHexString(0xFF33, 6); // results in "0x00FF33"

Returns
String — String representation of the number as a string starting with "0x"
