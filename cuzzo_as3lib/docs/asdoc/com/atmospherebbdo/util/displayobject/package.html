align()function
public function align(target:DisplayObject, bounds:HorizontalAlignment, hAlign:VerticalAlignment = null, vAlign:* = null):void

Adapted from Yahoo! Flash Developer utilities. Aligns a DisplayObject vertically and horizontally within specific bounds.

Parameters

target:DisplayObject — The DisplayObject to align.
 
bounds:HorizontalAlignment — something with x, y, height, and width: a rectangle or display object, usually.
 
hAlign:VerticalAlignment (default = null) — VerticalAlignment, the alignment position along the horizontal axis. If null, the target's horizontal position will not change.
 
vAlign:* (default = null) — HorizontalAlignment, the alignment position along the vertical axis. If null, the target's vertical position will not change.

resizeAndMaintainAspectRatio()function 
public function resizeAndMaintainAspectRatio(target:DisplayObject, width:Number, height:Number, aspectRatio:Number):void

Adapted from Yahoo! Flash Developer utilities. Resizes a DisplayObject to fit into specified bounds such that the aspect ratio of the target's width and height does not change.

Parameters

target:DisplayObject — The DisplayObject to resize.
 
width:Number — The desired width for the target.
 
height:Number — The desired height for the target.
 
aspectRatio:Number (default = NaN) — The desired aspect ratio. If NaN, the aspect ratio is calculated from the target's current width and height.
