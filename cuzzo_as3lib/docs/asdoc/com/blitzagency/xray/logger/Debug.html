public static function trace(pMsg:Object, pPackage:String, pLevel:Number = 0):void

Traces any value to the debug panel, with an optional message level.

Parameters

pMsg:Object — The value to trace.
 
pPackage:String — Optional. The level for this message. Values are 0 through 4, or Debug.Debug, Debug.INFO, Debug.WARN, Debug.ERROR, Debug.FATAL.
 
pLevel:Number (default = 0)

public static function traceObject(o:Object, pRecurseDepth:Number = 254, pIndent:Number = 0, pPackage:String, pLevel:Number = 0):void

Recursively traces an object's value to the debug panel.

Parameters

o:Object — The object to trace.
 
pRecurseDepth:Number (default = 254) — Optional. How many levels deep to recursively trace. Defaults to 0, which traces only the top level value.
 
pIndent:Number (default = 0) — Optional. Number of spaces to indent each new level of recursion.
 
pPackage:String — - passed in via XrayLogger. Package info sent along to Xray's interface for package filtering
 
pLevel:Number (default = 0)
