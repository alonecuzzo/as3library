InheritanceXray flash.display.Sprite

This is the main entry point for Xray

Xray needs to be added to a sprite to gain access to the stage property. The Flex2Xray class extends Xray and overrides the init method. The Flex2Xray instance does not have to added to any display list, just use the new constructor and it'll light right up
